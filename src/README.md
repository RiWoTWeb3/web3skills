## üöÄ Quick Start

### Prerequisites
- Node.js 16+ installed
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/RiWoT/web3skills.git
cd web3skills

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
npm start
```

## üéØ How to Use

### 1. **Track Your Skills**
- Navigate to the **Skills** tab
- Check off skills as you learn them
- Watch your progress grow automatically
- Filter by category or search for specific skills

### 2. **Explore Career Paths**
- Visit the **Careers** tab
- See your match percentage for each path
- Select a career to view detailed roadmap
- Follow the 4-phase learning plan

### 3. **Access Learning Resources**
- Each career phase includes curated resources
- Direct links to courses (Cyfrin Updraft, Turbin3, etc.)
- Mix of FREE and paid (earn money) resources
- Track which resources you've completed

### 4. **Join the Community**
- Connect with [RiWoT Discord](https://discord.gg/epWxxeWC)
- Contribute to [RiWoT GitHub](https://github.com/RiWoT)
- Get mentorship and code reviews
- Participate in learning sessions

### 5. **Manage Your Data**
- **Export**: Download your progress as JSON
- **Import**: Restore from a backup file
- **Share**: Generate a shareable code
- **View**: Load someone else's progress
