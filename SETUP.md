
  # Web3 Skills Learning Platform

  This is a code bundle for Web3 Skills Learning Platform. The original project is available at https://www.figma.com/design/jk3pkBkKU9bbc4damrYn4t/Web3-Skills-Learning-Platform.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
